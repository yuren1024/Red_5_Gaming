# 红五五家功能说明

本文档用于单独介绍当前项目中的“红五五家”模块，方便把项目总览与具体游戏说明分开维护。

## 模块定位

红五五家模块是一个面向 5 人局的本地记分与统计工具，重点不是牌局过程记录，而是对局结束后的快速结算、累计统计和历史复盘。

它当前支持：

- 一般情况
- 独打
- 逃跑
- 历史记录编辑与删除
- 总分、胜率、叫庄率统计
- CSV / JSON 导出

## 核心能力

### 玩家管理

- 固定 5 个玩家槽位，内部编号为 `A / B / C / D / E`
- 支持自定义玩家显示名
- 改名不会影响历史记录关联

### 单局录入

当前支持以下组合：

- 一般情况 + 抓分结算
- 独打 + 抓分结算
- 一般情况 + 逃跑
- 独打 + 逃跑

录入时页面会根据模式自动切换字段显示：

- 一般情况显示伴家
- 独打隐藏伴家
- 抓分结算显示闲家抓分
- 逃跑显示逃跑玩家

### 自动结算

系统会根据：

- 模式
- 结束方式
- 庄家
- 伴家
- 闲家抓分或逃跑玩家

自动推导：

- 闲家列表
- 胜方
- 五名玩家结算分
- 规则快照

### 数据统计

当前会实时计算：

- 总局数
- 当前领先玩家
- 总分排行榜
- 每位玩家总分
- 每位玩家胜率
- 每位玩家叫庄率

### 历史记录

- 支持查看所有历史对局
- 支持编辑与删除
- 修改记录后自动重算统计结果

### 导出

支持两种导出方式：

- `CSV`
- `JSON`

适合用于备份、发给牌友、或后续继续整理统计。

## 使用流程

1. 登录后进入 `games.html`
2. 点击“红五五家”进入主页面
3. 设置玩家姓名
4. 录入模式、结束方式、庄家等信息
5. 点击“预览结算”确认本局分数
6. 点击“保存本局”写入历史记录
7. 在排行榜和历史记录区域查看累计结果

## 规则摘要

### 一般情况

- 庄家 + 伴家，对 3 名闲家
- 闲家抓分 `< 120` 时庄家方胜
- 闲家抓分 `>= 120` 时闲家方胜

### 独打

- 庄家，对 4 名闲家
- 闲家抓分 `< 120` 时庄家胜
- 闲家抓分 `>= 120` 时闲家胜

### 逃跑

- 一般模式：逃跑者 `-80`，其余四家各 `+10`
- 独打模式：逃跑者 `-200`，其余四家各 `+25`
