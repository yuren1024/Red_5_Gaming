# 红五游戏中心

这是一个基于浏览器的本地游戏中心项目。目前已经不再是单一的红五五家记分工具，而是一个可以持续扩展的网页应用，当前提供红五五家记分、贪吃蛇与 2D 赛车计时赛三个入口。

## 项目定位

项目采用原生 HTML、CSS、JavaScript 实现，前端没有构建流程；当前新增了一个轻量 Node.js 服务端，用于登录鉴权与受保护页面访问控制。整体目标仍然是提供一个轻量、方便继续添加小游戏或工具页的本地游戏中心。

当前已提供：

- 红五五家记分与统计
- 贪吃蛇小游戏
- 2D 赛车计时赛
- 登录页与游戏选择页

## 当前游戏

### 红五五家

这是当前最完整的核心功能模块，适用于 5 人红五五家的本地记分、统计、历史记录与导出。

详细介绍见：

- `docs/hongwu-wujia-overview.md`
- `docs/hongwu-wujia-score-tool-design.md`

### 贪吃蛇

这是一个本地单机小游戏，支持：

- 键盘方向键和 `W / A / S / D`
- 移动端触控方向按钮
- 随分数提升的动态速度
- 最高分本地持久化

### 2D 赛车计时赛

这是一个俯视角单机计时赛车小游戏，支持：

- 键盘方向键控制赛车
- 直线高速与转弯自动减速
- 固定赛道上的 3 圈计时挑战
- 本地最佳单圈持久化

## 使用方式

### 登录

1. 打开 `login.html`
2. 使用管理员分配的账号登录
3. 登录后进入 `games.html` 选择具体游戏

### 运行项目

先启动认证服务，再通过浏览器访问登录页。

```bash
# 进入服务端目录
cd server

# 安装依赖
npm install

# 复制配置模板并填写管理员账号、密码哈希和会话密钥
cp .env.example .env

# 如果需要生成密码哈希
npm run hash-password -- "your-password"

# 启动服务
npm start
```

启动后访问 `http://localhost:3000/login.html`。

## 技术说明

- 前端：原生 `HTML + CSS + JavaScript`
- 存储：浏览器 `localStorage`
- 鉴权：Node.js 服务端会话鉴权
- 部署：Node.js 服务端托管页面与认证接口

项目中的不同页面按“入口页 + 独立脚本”组织，便于后续继续增加新的小游戏或工具页面。

## 数据与存储

当前数据均保存在浏览器本地，不会上传到服务器。

- 红五五家主数据使用独立的 `localStorage` 键保存
- 贪吃蛇最高分使用独立的 `localStorage` 键保存
- 赛车游戏最佳圈速与最近成绩使用独立的 `localStorage` 键保存
- 清除浏览器数据后，本地记录会一并丢失
