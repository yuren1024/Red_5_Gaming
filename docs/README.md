# 红五游戏中心

这是一个基于浏览器的本地游戏中心项目。目前已经不再是单一的红五五家记分工具，而是一个可以持续扩展的静态网页应用，当前提供红五五家记分与贪吃蛇两个入口。

## 项目定位

项目采用原生 HTML、CSS、JavaScript 实现，不依赖后端服务，也没有构建流程。当前目标是提供一个轻量、可直接打开使用、方便继续添加小游戏或工具页的本地游戏中心。

当前已提供：

- 红五五家记分与统计
- 贪吃蛇小游戏
- 登录页与游戏选择页

## 当前游戏

### 红五五家

这是当前最完整的核心功能模块，适用于 5 人红五五家的本地记分、统计、历史记录与导出。

详细介绍见：

- `docs/hongwu-wujia-overview.md`
- `docs/hongwu-wujia-score-tool-design.md`

### 贪吃蛇

这是一个本地单机小游戏，支持：

- 键盘方向键和 `W / A / S / D`
- 移动端触控方向按钮
- 随分数提升的动态速度
- 最高分本地持久化

## 使用方式

### 登录

1. 打开 `login.html`
2. 使用默认账号登录
3. 登录后进入 `games.html` 选择具体游戏

默认账号：

- 账号：`于人`
- 密码：`123456`

### 运行项目

直接打开页面即可，或使用一个本地静态服务器。

```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Node.js http-server
npx http-server .
```

启动后访问 `http://localhost:8000/login.html`。

## 技术说明

- 前端：原生 `HTML + CSS + JavaScript`
- 存储：浏览器 `localStorage`
- 鉴权：本地固定账号校验
- 部署：静态文件直接运行

项目中的不同页面按“入口页 + 独立脚本”组织，便于后续继续增加新的小游戏或工具页面。

## 数据与存储

当前数据均保存在浏览器本地，不会上传到服务器。

- 红五五家主数据使用独立的 `localStorage` 键保存
- 贪吃蛇最高分使用独立的 `localStorage` 键保存
- 清除浏览器数据后，本地记录会一并丢失
